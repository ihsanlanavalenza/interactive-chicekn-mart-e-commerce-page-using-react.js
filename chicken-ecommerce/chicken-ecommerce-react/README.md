## 🚀 **Quick Start**

### Prerequisites
- Node.js 16+ 
- npm atau yarn

### Installation

```bash
# 1️⃣ Clone repository
git clone https://github.com/yourusername/chicken-ecommerce-react.git
cd chicken-ecommerce-react

# 2️⃣ Install dependencies
npm install

# 3️⃣ Start development server
npm start

# 4️⃣ Open browser
# Navigate to http://localhost:3000
```

### Build untuk Production

```bash
# Build optimized version
npm run build

# Serve production build locally
npx serve -s build
```

## 🔧 **Configuration & Customization**

<summary><strong>🎨 Theme Customization</strong></summary>

Ubah warna tema utama di CSS:
```css
:root {
  --primary-color: #ff6b35;
  --secondary-color: #f7931e;
  --accent-color: #ffeb3b;
  --text-color: #333;
```

</details>

<details>
<summary><strong>📊 Chart Configuration</strong></summary>

Kustomisasi chart di `Analytics.js`:
```javascript
const chartOptions = {
  responsive: true,
  maintainAspectRatio: false,
  // Add your custom options
};
```
